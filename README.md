# Brewer::Server

This gem is a web interface for [llamicron/brewer](http://github.com/llamicron/brewer)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'brewer-server'
```

And then execute:

```shell
$ bundle
```

Or install it yourself as:

```shell
$ gem install brewer-server
```

## Usage
Run

```shell
$ brewer-server
```

Visit `localhost:8080`
